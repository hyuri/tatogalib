How to use a template
=====================

1. Copy the template folder to its target parent folder
2. Rename the package folder
3. Rename the class name
4. Get the latest Toga code
5. Use the Toga code to populate the new package
6. Test if the new (unchanged) package still works
7. Make a commit, so you can see the changes to did
8. Make your custom changes
9. Commit and push
